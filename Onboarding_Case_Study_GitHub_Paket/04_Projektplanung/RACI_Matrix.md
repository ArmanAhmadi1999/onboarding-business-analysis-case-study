# RACI-Matrix

| Aufgabe | HR | Teamlead | Buddy / erfahrene Mitarbeitende | Neuer Mitarbeitender | IT |
|---|---|---|---|---|---|
| Onboarding-Plan erstellen | A/R | C | C | I | I |
| Inhalte & Dokumente bereitstellen | A/R | R | C | I | I |
| Fachliche Einarbeitung planen | C | A/R | R | I | I |
| Feedbackgespräche durchführen | R | A/R | R | R | I |
| Plattformzugang einrichten | I | I | I | I | A/R |
| 30-Tage-Selbstcheck durchführen | I | C | C | R | I |
| Ergebnisse auswerten & verbessern | A/R | C | C | I | I |

R = Responsible, A = Accountable, C = Consulted, I = Informed
